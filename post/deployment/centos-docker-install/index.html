CentOS 生产环境安装 Docker

2019-08-16
服务器部署

Contents


1

$ yum info docker

CentOS 提供的 docker 默认版本为 1.13.1，版本较低。

我们来安装 docker-ce

安装 yum-utils

1

$ yum install -y yum-utils

添加 yum 源

1
2
3

$ yum-config-manager \
    --add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo

更新yum的包索引

1

$ yum makecache fast

安装docker

1

$ yum install docker-ce

查看版本

1
2

$ docker -v
Docker version 19.03.1, build 74b1e89

启动 docker 服务

1
2

$ service docker start
# chkconfig docker on

提示

权限不足

显示没有权限

1

$ sudo chmod 666 /var/run/docker.sock

为 docker.sock 添加权限666，解决。
