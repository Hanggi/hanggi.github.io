官网

安装

Flutter 支持 Windows、Mac 和 Linux。

Mac下安装Flutter

官方 Mac版 安装教程

1.下载

可在官网点击下载，

也可以通过下列命令，下载最新稳定版本

1

$ wget -P ~/Downloads https://storage.googleapis.com/flutter_infra/releases/stable/macos/flutter_macos_v1.7.8+hotfix.4-stable.zip

文件稍大，大概 500MB 多，请耐心下载。

2.解压

解压到你想要的位置

1
2

$ cd ~/development
$ unzip ~/Downloads/flutter_macos_v1.7.8+hotfix.4-stable.zip

3.添加环境变量

1

$ export PATH="$PATH:`pwd`/flutter/bin"

Xcode - develop for iOS and macOS

这里按照提示安装Xcode最新版并安装或升级cocoapods

1
2

$ brew upgrade cocoapods
$ pod setup

过程可能稍长，请耐心等待。

iOS tools - develop for iOS devices

这一步只要按提示输入命令即可：

1
2
3
4

6
7

$ brew update
$ brew install --HEAD usbmuxd
$ brew link usbmuxd
$ brew install --HEAD libimobiledevice
$ brew install ideviceinstaller

$ brew install ios-deploy

VS Code

如果不使用 VS Code 这一项将不会影响我们开发，但是为了强迫症患者，我们安装一下。

方法很简单，进到 VS Code 插件市场搜索 Flutter 下载即可。
